Make a fun programming language. Get $40 for electronics!

What is this?

This is a YSWS that is not about building a productive programming language, but about building a programming language that is fun and creative - an esolang!

What are the requirements?

First off: you can have a lot of fun designing your language however you want, but we do have some requirements to make sure it works! It should feature:

  • Variables
  • Operators
  • Functions
  • Error Handling

What do I get?

You get a $40 grant to buy electronics from our approved list!
