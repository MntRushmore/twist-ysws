Make a fun programming language and get $40 to buy electronics!

What is this?

This is a YSWS that is not about building a productive programming language, but about building a programming language that is fun and creative - an esolang!

What are the requirements?

You can have a lot of fun designing your language however you want, but we do have some requirements to make sure it works! It should feature:

  • Variables
  • Operators
  • Functions
  • Error Handling
